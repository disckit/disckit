## About

This repository contains multiple packages for building Discord bots and dashboards. You can find the main entry package at [`disckit`](./disckit). It is a collection of zero-dependency Node.js utilities that work in bots, REST APIs and React dashboards.

## Packages

- [disckit](disckit) ([source](disckit/src/index.js)) - Install all @disckit packages at once
- [@disckit/paginator](paginator) ([source](paginator/src/index.js)) - Universal pagination engine for bots and dashboards
- [@disckit/i18n](i18n) ([source](i18n/src/index.js)) - Lightweight i18n with interpolation, pluralization and fallback locale
- [@disckit/permissions](permissions) ([source](permissions/src/index.js)) - Human-readable Discord permission bitfields, no discord.js required
- [@disckit/cooldown](cooldown) ([source](cooldown/src/index.js)) - Per-user, per-guild, per-command cooldown manager for bots

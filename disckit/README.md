## About

`disckit` is the main entry point — install all `@disckit` packages at once.

Node.js 18.0.0 or newer is required.

## Installation

```sh
npm install disckit
yarn add disckit
pnpm add disckit
```

Or install packages individually:

```sh
npm install @disckit/paginator
npm install @disckit/i18n
npm install @disckit/permissions
npm install @disckit/cooldown
```

## Packages

- [@disckit/paginator](../paginator) - Universal pagination engine for bots and dashboards
- [@disckit/i18n](../i18n) - Lightweight i18n with interpolation, pluralization and fallback locale
- [@disckit/permissions](../permissions) - Human-readable Discord permission bitfields, no discord.js required
- [@disckit/cooldown](../cooldown) - Per-user, per-command cooldown manager for bots

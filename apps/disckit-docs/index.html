disckit is a collection of zero-dependency Node.js utilities for building Discord bots, REST APIs and React dashboards. Ships with CJS + ESM and full TypeScript types.

Docs Guide ↗ GitHub ↗
> npm install disckit
@disckit/common
Strings, arrays, time and async utilities
@disckit/antiflood
Token bucket rate limiting for bots
@disckit/cooldown
Per-user, per-command cooldown manager
@disckit/paginator
Universal pagination for bots and dashboards
@disckit/permissions
Human-readable Discord permission bitfields
@disckit/placeholders
Placeholder engine for welcome messages
@disckit/i18n
i18n with interpolation and pluralization
@disckit/cache
LRU and TTL in-memory caches
@disckit/caffeine
Async cache builder with background refresh

About #

disckit is a powerful Node.js module for building Discord bots and React dashboards. It takes a modular, zero-dependency approach.

  • Zero external dependencies
  • Dual CJS + ESM support
  • Full TypeScript definitions included
  • Works in Node.js 18+, bots, REST APIs and React

Installation #

Node.js 18.0.0 or newer is required.

Shell 
npm install disckit
yarn add disckit
pnpm add disckit
bun add disckit

Packages #

Install the meta-package to get everything, or pick individual packages:

  • @disckit/common — formatTime, sleep, chunk, randomInt, isHexColor and more
  • @disckit/antiflood — token bucket rate limiting for bots
  • @disckit/cooldown — per-user, per-guild, per-command cooldowns
  • @disckit/paginator — universal pagination engine for bots and dashboards
  • @disckit/permissions — human-readable Discord permission bitfields, no discord.js required
  • @disckit/placeholders — placeholder engine for welcome messages and bot status
  • @disckit/i18n — lightweight i18n with interpolation and pluralization
  • @disckit/cache — LRU and TTL in-memory caches with O(1) get/set
  • @disckit/caffeine — Caffeine-inspired async cache builder with background refresh

Example usage #

These examples use ES modules.

Cooldown manager

index.js
import { CooldownManager } from 'disckit';

const cooldowns = new CooldownManager({ default: 3000 });

client.on('interactionCreate', async interaction => {
  if (!interaction.isChatInputCommand()) return;

  const result = cooldowns.check(interaction.commandName, interaction.user.id);
  if (!result.ok) {
    return interaction.reply({
      content: `⏳ Wait ${result.remainingText} before using this again.`,
      ephemeral: true,
    });
  }

  // run the command...
});

Paginator

leaderboard.js
import { paginate } from 'disckit';

const members = await db.getTopMembers();
const page = paginate(members, { limit: 10, page: 1 });

// page.items   → first 10 entries
// page.totalPages  → total page count
// page.hasNext → true if more pages

Permissions

permissions.js
import { Permissions } from 'disckit';

const perms = new Permissions(['SEND_MESSAGES', 'EMBED_LINKS']);

perms.has('SEND_MESSAGES'); // → true
perms.missing(['BAN_MEMBERS']); // → ['BAN_MEMBERS']

const admin = new Permissions('ADMINISTRATOR');
admin.has('BAN_MEMBERS'); // → true (ADMINISTRATOR bypasses)

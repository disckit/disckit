Introduction

disckit/disckit ⭐ Stars on GitHub

If you're reading this, it probably means you want to learn how to build a bot or dashboard using disckit. Awesome! You've come to the right place. This guide will teach you things such as:

  • How to get a bot up and running from scratch;
  • How to add cooldowns, rate limiting, and pagination properly;
  • In-depth explanations for caching, i18n, placeholders and permissions;
  • Working with TypeScript and both ESM and CJS environments;
  • And much more.

This guide will also cover common errors and how to solve them, keeping your code clean, and setting up a proper development environment. Sounds good? Great! Let's get started.

Before you begin...

Alright, building a bot is cool and all, but there are some prerequisites. To create a bot with disckit, you should have a fairly decent grasp of JavaScript itself. While you can build a bot with very little knowledge, trying to do so without understanding the language first will only hinder you — you may get stuck on uncomplicated issues and struggle with solutions to incredibly easy problems.

INFO

disckit works best with Node.js 18 or newer. Make sure you have the right version before starting.

If you don't know JavaScript but would like to learn it, here are a few links to get started:

  • Eloquent JavaScript, a free online book
  • JavaScript.info, a modern javascript tutorial
  • Codecademy's interactive JavaScript course
  • Nodeschool, for both JavaScript and Node.js lessons
  • MDN's JavaScript guide and full documentation
  • Google, your best friend

Take your pick, learn some JavaScript, and once you feel confident enough to build a bot, come back!

Quick example

Here's a minimal bot using disckit — cooldown management and rate limiting in under 30 lines:

index.js
import { Client, GatewayIntentBits } from 'discord.js';
import { CooldownManager, AntifloodManager, isBlocked, formatRetryAfter } from 'disckit';

const client = new Client({ intents: [GatewayIntentBits.Guilds] });
const cooldowns = new CooldownManager({ default: 3000 });
const antiflood = new AntifloodManager({
  globalRule: { windowMs: 5000, maxHits: 5, penaltyMode: 'NONE' },
});

client.on('interactionCreate', async interaction => {
  if (!interaction.isChatInputCommand()) return;

  const flood = antiflood.check({
    userId: interaction.user.id,
    guildId: interaction.guildId ?? '*',
    commandName: interaction.commandName,
    memberRoleIds: [],
  });
  if (isBlocked(flood)) {
    return interaction.reply({ content: `Slow down! Retry in ${formatRetryAfter(flood.retryAfterMs)}.`, ephemeral: true });
  }

  const cd = cooldowns.check(interaction.commandName, interaction.user.id);
  if (!cd.ok) {
    return interaction.reply({ content: `⏳ Wait ${cd.remainingText}!`, ephemeral: true });
  }

  // handle command...
});

client.login(process.env.BOT_TOKEN);
TIP

You can install all packages at once with npm install disckit, or install them individually for smaller bundles.
